# 📊 Social Media Analytics Tool

## 🚀 Overview

The **Social Media Analytics Tool** is a web application developed using Python and Flask that analyzes social media data and provides meaningful insights such as engagement metrics, trends, and performance analysis.

This project demonstrates my ability to build backend-driven applications, process data, and present it through a user-friendly interface.

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Python
* Flask

**Libraries & Tools:**

* Pandas (for data processing)
* Matplotlib / Chart.js (for visualization)

---

## ✨ Features

* 📊 Dashboard for analytics visualization
* 📈 Engagement analysis (likes, comments, shares)
* 📉 Trend analysis of posts
* 📁 Data processing using datasets
* 📱 Responsive user interface
* 🔄 Dynamic data rendering

## 📦 Installation & Setup

### 1️⃣ Clone the repository

```bash id="9lb8kb"
git clone https://github.com/bathulachandana/social-media-analytics-tool.git
cd social-media-analytics-tool
```

### 2️⃣ Install dependencies

```bash id="o4u8po"
pip install -r requirements.txt
```

### 3️⃣ Run the application

```bash id="zzv8tk"
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000/
```
